Building a Simple Shopping Cart: Create an array to store items added to the cart, with properties for product name, price, and quantity. Implement functionality to calculate the total cost, add and remove items, and update quantities.
  • Left Click on webpage. and select inspect option.
  • Select console in top right Navbar
    1. To see the Items in the cart (aaray): "cart"
    2. To add Item in the cart: "addItem(productionName.price,quantity) example: "addItem("Production 1",11,9);"
    3. To remove Item from the cart: "removeItem(index);" example: "removeItem(0);"
    4. to Update the quantity of the item in the cart: "updateQuantity(index, newQuantity);" example : "updateQuantity(1, 5);"
