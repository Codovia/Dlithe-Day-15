Creating a Playlist:Create an array of song objects with title, artist, and duration properties. Allow users to add, remove, and shuffle songs using array methods.

  • Left Click on webpage. and select inspect option.
  • Select console in top right Navbar
    1. To get the all the music type "songs" in console.
    2. To add Songs use this format in console : "addSong(title,artist,duration);" in the place of title,artist,duration fill your faviorite song details.
    3. To remove Songs use this format in console: "removeSong(index);" in the place of index type the index of the song yo want to remove from the array.
    4. To shuffle the music in the array call this function inthe console: "shuffleSongs();"
