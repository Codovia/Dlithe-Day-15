Managing a Contact Book: Create an array of objects representing contacts, with properties for name, email, phone number, and address. Allow users to search for contacts by name or email, add new contacts, and edit existing contact information.

  • Left Click on webpage. and select inspect option.
  • Select console in top right Navbar
    1. To search contact with email or name : "searchContact(query);" example: searchContact("John Doe");
    2. To add contact : "addContact(name, email, phone, address);" example : addContact("Harsha J", "Harsha@example.com","9638527410", "159 Manipal st")
    3. To edit contact email : "editContactEmail(index, newEmail);" example: " editContactEmail(1,"janes@example.com" )"
