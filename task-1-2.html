Implementing a To-Do List: Create an array of objects representing tasks, including a description, priority level, and completion status. Allow users to add, mark as complete, and filter tasks based on different criteria
  • Left Click on webpage. and select inspect option.
  • Select console in top right Navbar
    1. To see the task write "tasks" in the console
    2. To add the task write in this format "addTask(description,priority)" example: "addTask("Buy Book","High");
    3. To mark the task is completed write in this for mat "markAsCompleted(index)" example: "markAsCompleted(3);"
    4. To filter the task by priority: "filterTasksByPriority("priority")" example: "filterTasksByPriority("Medium")"
